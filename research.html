Research

Metabolic Network Analysis and Regulatory Pattern Searches

We develop and maintain software for overlaying multi-omic data upon the metabolic network and using that data to find interesting regulatory patterns within the data.

Metaboverse
  Status: Projects

Metaboverse: Metabolic network pattern recognition tools contextualize multi-omic data and reveal disease-relevant signatures

Description

Metaboverse is a cross-platform app built to aid users in contextualizing their data on their model’s metabolic and global reaction network. Metaboverse is an interactive tool for exploratory data analysis that searches user data in the context of the metabolic network to identify interesting patterns and trends within the data. Metaboverse aids users in interactively identifying interesting patterns and regulatory hotspots within their data for further experimental follow-up.

Protein-Metabolite Interaction Data Visualization and Analysis

We develop tools and models for identifying and contextualizing important protein-metabolite interaction regulatory events.

Electrum
  Status: Projects

Electrum: An online portal to visualizing and interpreting MIDAS protein-metabolite interaction data

Description

Electrum is an online interface for exploring data from the MIDAS database. MIDAS is a mass spectrometry platform developed by the Rutter Lab for the high-throughput identification of protein-metabolite interactions. Electrum provides visualizations of MIDAS data and interactive analytical tools.

Ribosome Profiling

We develop and maintain software that aids in the uniform and automated processing of ribosome profiling sequencing data.

XPRESSyourself
  Status: Projects

XPRESSyourself: Enhancing, Standardizing, and Automating Ribosome Profiling Computational Analyses Yields Improved Insight into Data

Description

XPRESSpipe is an automated, efficient, and flexible pipeline for end-to-end processing of ribosome profiling data. XPRESSpipe is currently capable of handling single-end (SE), paired-end (PE), and ribosome profiling data.
